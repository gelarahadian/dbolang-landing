# Warung Teh Ufu — Website

Website modern untuk Warung Teh Ufu, Cicurug, Sukabumi.

## Cara Menjalankan

```bash
# Install dependencies
npm install

# Jalankan development server
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

## Fitur
- 🏠 **Ringkasan** — Info cepat, rating chart, & kutipan ulasan
- 🍜 **Menu** — Daftar menu andalan dengan emoji & deskripsi
- ⭐ **Ulasan** — Tampilan ulasan lengkap dengan tombol like interaktif
- 📋 **Tentang** — Alamat, jam operasional, dan layanan
- 📱 Responsive mobile-first
- 🌿 Desain natural/organic dengan palet warna hijau
